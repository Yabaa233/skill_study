Dice Skill
一个用于 ClawPet Skill Registry 测试的骰子 Skill。

描述
掷骰子！支持多种模式：

标准骰子 — 掷 1 个 6 面骰子（d6）
自定义面数 — 指定骰子面数（如 d20、d100）
多颗骰子 — 一次掷多颗骰子
特殊骰子 — 骰宝风格（1-3 小 / 4-6 大）
触发场景
用户说以下任意内容时触发：

"掷骰子" / "摇骰子" / "roll dice"
"掷个 d20" / "roll d100"
"掷 3 颗骰子"
"来把骰宝"
使用方式
标准模式（d6）
输入: "掷骰子"
输出: 🎲 你掷出了 **5** 点！
自定义面数
输入: "掷个 d20"
输出: 🎲 d20 结果：**17**
多颗骰子
输入: "掷 3 颗骰子"
输出: 🎲 3d6 结果：[2, 5, 1] = **8**
骰宝模式
输入: "来把骰宝"
输出: 🎲 骰宝结果：**4** → 大！
脚本
scripts/roll.ps1 — PowerShell 骰子引擎（主脚本）
技术要求
OS: win32, darwin, linux
依赖: 无（纯 PowerShell / bash）
